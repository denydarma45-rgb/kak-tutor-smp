💡 Belum memiliki API Key? Anda bisa mendapatkannya secara GRATIS:

  1. Kunjungi situs resmi Google AI Studio.
  2. Silakan masuk (Login) menggunakan akun Google atau Gmail Anda.
  3. Klik menu Get API Key, lalu pilih tombol Create API Key.
  4. Salin (Copy) kode panjang yang diberikan, kemudian tempelkan (Paste) ke dalam kotak input di atas.
